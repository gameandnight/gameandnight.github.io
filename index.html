Suite de Mini Juegos controlados con gestos (OpenCV + Tkinter)

Aplicación en Python que ofrece varios mini juegos interactivos controlados mediante gestos o pose corporal, usando OpenCV y una interfaz Tkinter.

Descripción general / Pipeline:

  • Launcher en Tkinter con botones para cada mini juego: "Manos", “Cuerpo”, “Dibujar”, “Atrapa la pelota”, “Piedra, Papel o Tijera” y “Snake”.
  • Para detección de landmarks de mano y pose se emplea CVZone PoseModule/HandTracking, facilitando el uso de MediaPipe para extraer coordenadas clave.
  • Lógica de cada mini juego: captura de cámara con cv2.VideoCapture, procesamiento de frames, interpretación de gestos para mover sprites, dibujar o decidir jugadas.
  • Manejo de eventos de teclado (‘q’ para salir) y retorno al menú sin cerrar la app por completo.

Recomendador Semántico — Catálogo con búsqueda híbrida

Demo de un motor de búsqueda híbrido para catálogo de productos que combina embeddings (FAISS) y BM25, con opción de re-ranking por Cross-Encoder. Backend en FastAPI y frontend SPA en React + Vite.

  • Stack:
    • Python 3.10 — FastAPI
    • Sentence-Transformers, FAISS (faiss-cpu) — búsqueda semántica por vectores
    • rank_bm25 — búsqueda textual
    • React + Vite para UI; despliegue estático / demo
  • Características clave:
    • Búsqueda híbrida — unión de candidatos FAISS + BM25 y mezcla de scores.
    • Boost por coincidencia textual (Jaccard + stemming) y normalización softmax.
    • Reranker opcional (cross-encoder) para ajustar el ranking final.
    • Generación automática de corpus tokenizado si no existe (activa BM25).
    • Interfaz responsive con controles (α/β, k, re-rank).
  • ¿Cómo probarlo?
    • Clona el repo: git clone https://github.com/gameandnight/Recomendador-semantico
    • Backend: crea un entorno Python, instala dependencias y arranca con: 
      uvicorn backend.app.main:app --reload --port 8000
    • Frontend: cd frontend && npm install && npm run dev
